# Vault Source Plugin

The Vault Source plugin for CloudQuery extracts configuration from a variety of Vault APIs and loads it into any supported CloudQuery destination (e.g. PostgreSQL, BigQuery, Snowflake, and [more](https://www.cloudquery.io/docs/plugins/destinations/overview)).
