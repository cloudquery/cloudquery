# Tailscale Plugin

The CloudQuery Tailscale plugin pulls configuration out of Tailscale and loads it into any supported CloudQuery [destination](https://www.cloudquery.io/docs/plugins/destinations/overview) (e.g. PostgreSQL, BigQuery, Snowflake, and [more](https://www.cloudquery.io/docs/plugins/destinations/overview)).
