The CloudQuery Google Analytics plugin allows to run custom
[Google Analytics Data API v1 reports](https://developers.google.com/analytics/devguides/reporting/data/v1/basics#reports)
and loads it into any supported CloudQuery destination (e.g. PostgreSQL, BigQuery, Snowflake, and [more](/docs/plugins/destinations/overview)).
