You can download the precompiled binary from [releases](https://github.com/cloudquery/cloudquery/releases), or using CLI:

<Tabs options={['OSX', 'Linux', 'Windows']}>

<Tab>
<Tabs options={['Homebrew', 'Precompiled Binaries (amd64)', 'Precompiled Binaries (arm64)']}>
<Tab>

```bash copy
brew install cloudquery/tap/cloudquery

# After initial install you can upgrade the version via:
# brew upgrade cloudquery
```

</Tab>

<Tab>

```bash copy
curl -L https://versions.cloudquery.io/latest/v2/cloudquery_darwin_amd64 -o cloudquery
chmod a+x cloudquery
```

</Tab>

<Tab>

```bash copy
curl -L https://versions.cloudquery.io/latest/v2/cloudquery_darwin_arm64 -o cloudquery
chmod a+x cloudquery
```

</Tab>

</Tabs>
