# Table: facebookmarketing_adplayables

This table shows data for Facebook Marketing Ad Playables.

The composite primary key for this table is (**account_id**, **id**).

|_cq_sync_time|Timestamp|
|_cq_id|UUID|
|_cq_parent_id|UUID|

|id (PK)|String|
|name|String|
|owner|JSON|
