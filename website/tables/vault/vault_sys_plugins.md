# Table: vault_sys_plugins

This table shows data for Vault Sys Plugins.

The primary key for this table is **_cq_id**.

## Columns

| Name          | Type          |
| ------------- | ------------- |
|_cq_id (PK)|`uuid`|
|_cq_parent_id|`uuid`|
|type|`utf8`|
|name|`utf8`|
|version|`utf8`|
|builtin|`bool`|
|deprecation_status|`utf8`|
