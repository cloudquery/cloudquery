# Table: cloudflare_ips

This table shows data for Cloudflare IPs.

The primary key for this table is **_cq_id**.

| Name          | Type          |
| ------------- | ------------- |
|_cq_source_name|String|
|_cq_sync_time|Timestamp|
|_cq_id (PK)|UUID|
|_cq_parent_id|UUID|
|ip|String|
|type|String|
